Tentang Saya

Kenali lebih jauh tentang latar belakang, pengalaman, dan passion saya terhadap teknologi

Mahasiswa Rekayasa Perangkat Lunak yang Bersemangat

Saya adalah mahasiswa semester 7 program studi Rekayasa Perangkat Lunak di Telkom University Purwokerto dengan minat tinggi pada pengembangan antarmuka (UI/UX), pemrograman front-end, dan pengembangan aplikasi mobile menggunakan Flutter.

Terbiasa bekerja dengan desain modern menggunakan Figma serta membangun sistem berbasis web responsif menggunakan Tailwind CSS. Saya selalu antusias untuk belajar teknologi baru dan berkontribusi dalam proyek-proyek yang menantang.

Dengan pengalaman dalam berbagai teknologi seperti React.js, Angular, Next.js, dan Flutter, saya siap untuk menghadapi tantangan dalam dunia pengembangan perangkat lunak dan berkontribusi dalam tim yang dinamis.
